## Overview

This is a fully responsive HTML email newsletter designed for **Kasha Decor**, a fictional interior decor brand. Built from scratch using email-safe best practices, it showcases clean, modular code for marketing campaigns, promotions, and customer engagement emails.

## Key Features

- **Mobile-First Responsive Design**: Uses fluid tables, media queries, and hybrid techniques to ensure perfect rendering on all screen sizes.
- **Cross-Client Compatibility**: Table-based structure with inline CSS for reliable display in major email clients (Gmail, Outlook, Apple Mail, Yahoo, etc.).
- **Email-Safe Coding**: No external CSS/JS dependencies; all styles inlined where needed. Supports dark mode basics and fallback fonts.
- **Modular Sections**: Easy to customizeâ€”hero banner, featured products, discount code spotlight, and footer with social links/unsubscribe.
- **Performance Optimized**: Lightweight code, alt text on images, and preheader text for better inbox preview.

## Technologies Used

- HTML5
- CSS3 (inline and embedded for email compatibility)
- Responsive techniques (media queries, fluid widths)

## How to Use / Customize

1. Clone the repo: `git clone https://github.com/teboho-xaba/KashaDecorNewsletter.git`
2. Open `index.html` in your browser for preview.
3. Edit sections directly in the HTML (e.g., update images, text, links).
4. For production: Use an inliner tool (like Premailer) and test with Litmus/Email on Acid/MintyEmailTesters.
5. Import into ESPs like Mailchimp, Klaviyo, or SendGrid.

## Testing

- Basic rendering checks in Gmail, Outlook, and Apple Mail.
